# VS Code Setup

## Entorno de desarrollo

Editor utilizado: Visual Studio Code

Sincronización de configuraciones: Activada mediante cuenta de GitHub.

Esto permite sincronizar entre dispositivos:

* Configuración
* Extensiones
* Atajos de teclado
* Fragmentos de código
* Tareas

---

## Extensiones instaladas

Las siguientes extensiones fueron instaladas para el entorno de desarrollo:

* GitHub Copilot Chat
* GitHub Pull Requests & Issues
* GitLens
* Markdown All in One
* Prettier
* REST Client

---

## Descripción de extensiones

**GitHub Copilot Chat**
Asistente de inteligencia artificial integrado en el editor para generación y explicación de código.

**GitHub Pull Requests & Issues**
Permite gestionar Pull Requests e Issues de GitHub directamente desde VS Code.

**GitLens**
Extensión avanzada para visualizar historial de Git, autores de cambios y navegación del repositorio.

**Markdown All in One**
Mejora la edición de archivos Markdown con atajos, vista previa y herramientas de formato.

**Prettier**
Formateador automático de código para mantener consistencia en el estilo del proyecto.

**REST Client**
Permite realizar pruebas de APIs HTTP directamente desde archivos `.http` o `.rest`.

---

## Notas

La configuración del editor está sincronizada mediante Settings Sync de VS Code, lo que permite replicar el entorno fácilmente en otros dispositivos.
